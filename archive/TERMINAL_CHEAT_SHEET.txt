## ‚ö° POWER USER TRICKS

### Schnelle Navigation
```cmd
cd ~                            # Zu User Home (C:\Users\username\)
cd %USERPROFILE%                # Auch zu User Home
cd %APPDATA%                    # Zu AppData Ordner
```

### Tab-Completion
- Anfang tippen + **TAB** dr√ºcken = Auto-Complete! üéØ
- Beispiel: `cd Pro` + TAB ‚Üí `cd Projects\`

### History durchsuchen
- **Pfeil hoch/runter** = Durch Commands scrollen
- **F7** = Zeigt Command-History als Liste!

### Command abbrechen
- **Ctrl + C** = Stoppt laufenden Command/Programm
- **Ctrl + Z** (dann Enter) = EOF Signal

### Copy & Paste im Terminal
- **Rechtsklick** = Paste (in neueren Windows Terminals)
- **Ctrl + Shift + C** = Copy (in Windows Terminal)
- **Ctrl + Shift + V** = Paste (in Windows Terminal)

## üìä VERGLEICH: DOS vs. HEUTE

| Funktion | DOS (1990s) | Windows CMD (heute) | Git Bash | PowerShell |
|----------|-------------|---------------------|----------|------------|
| List files | `dir` | `dir` | `ls` | `dir` / `ls` |
| Change dir | `cd` | `cd` | `cd` | `cd` |
| Clear | `cls` | `cls` | `clear` | `cls` / `clear` |
| Copy | `copy` | `copy` | `cp` | `Copy-Item` |
| Delete | `del` | `del` | `rm` | `Remove-Item` |

**Du kannst auch:** Git Bash verwenden (kommt mit Git) - das hat Unix-Commands wie `ls`, `rm`, etc.
